# CardSum Lens 操作マニュアル

## 1. アプリ概要

CardSum Lens は、QRコードを印刷または表示したカードをカメラで読み取り、カードに含まれる数値を自動で合計するWebアプリです。サーバーは不要で、ブラウザだけで動作します。

## 2. 利用環境

- Windows: Chrome、Edge などの最新版
- Chromebook: Chrome 最新版
- カメラ付き端末
- GitHub Pages などのHTTPS環境

カメラ機能はブラウザのセキュリティ制限により、HTTPSまたはlocalhostでのみ動作します。

## 3. QRコードの作り方

QRコードに入れる文字列は、次の形式にしてください。

```text
カードID,数値
```

例:

```text
card001,100
card002,25
card003,50
```

カードIDは重複防止に使います。同じカードを複数回かざしても、同じカードIDは1回だけ登録されます。

## 4. 基本操作

1. アプリを開きます。
2. `カメラ開始` を押します。
3. ブラウザにカメラ使用の確認が出たら許可します。
4. QRコードを画面中央の枠に入るようにかざします。
5. 読み取りに成功すると、カードID、数値、読み取り時刻が表に追加されます。
6. 画面右側の合計、枚数、平均が自動更新されます。

## 5. 手入力テスト

カメラを使わずに動作確認したい場合は、`手入力テスト` に `card001,100` のような文字列を入力して `追加` を押します。

## 6. データ操作

- `削除`: 対象カードだけを削除します。削除後は同じカードIDを再登録できます。
- `リセット`: 読み取り済みカードをすべて消去します。
- `CSV保存`: 現在の読み取り結果をCSVファイルとして保存します。

## 7. トラブル対応

### カメラが開始できない

- GitHub PagesなどのHTTPS URLで開いているか確認してください。
- ブラウザのカメラ許可がブロックされていないか確認してください。
- ほかのアプリがカメラを使用していないか確認してください。

### QRコードを読み取れない

- QRコードを画面中央の枠内に入れてください。
- 画面や紙の反射を避けてください。
- QRコード内の文字列が `カードID,数値` 形式になっているか確認してください。

### 同じカードが追加されない

これは重複防止機能です。同じカードIDは1回だけ登録されます。再登録したい場合は該当行の `削除` を押すか、`リセット` してください。

## 8. 教材・授業での運用メモ

- 最初は手入力テストで計算ロジックを確認してください。
- 次にスマートフォン画面へQRコードを表示し、読み取りテストを行ってください。
- 最後に紙へ印刷し、WindowsとChromebookの両方で動作確認してください。
