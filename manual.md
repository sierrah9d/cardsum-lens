# CardSum Lens 操作マニュアル

## 1. アプリ概要

CardSum Lens は、QRコードを印刷または表示したカードをカメラで読み取り、カードに含まれる栄養情報を自動で集計する Web アプリです。サーバーは不要で、ブラウザだけで動作します。

## 2. 利用環境

- Windows: Chrome、Edge などの最新版
- Chromebook: Chrome 最新版
- カメラ付き端末
- GitHub Pages などの HTTPS 環境

カメラ機能はブラウザのセキュリティ制限により、HTTPS または localhost でのみ動作します。

## 3. QRコードの作り方

QRコードに入れる文字列は、次の形式にしてください。

```text
カードID,名称,グラム数,カロリー,炭水化物,タンパク質,脂質,食物繊維,カリウム,カルシウム,鉄,ビタミンA,ビタミンC,ビタミンD,食塩
```

例:

```text
card001,rice,150,234,55.7,3.8,52,1200,230,45,1.2,310,35,5.5,0.4
card002,bread,80,210,40.2,6.5,48,900,120,25,0.8,0,0,0,0.6
```

カードIDは重複防止に使います。同じカードを複数回かざしても、同じカードIDは1回だけ登録されます。

## 4. 基本操作

1. アプリを開きます。
2. `カメラ起動` を押します。
3. ブラウザにカメラ使用の確認が出たら許可します。
4. QRコードを画面中央の枠に入るようにかざします。
5. 読み取りに成功すると、カードID、名称、栄養情報、読み取り時刻が表に追加されます。
6. 右側の栄養サマリーが自動更新されます。

読み取り成功時は短い成功音が鳴ります。QRコードは読めたが形式が不正な場合は失敗音が鳴ります。

## 5. 栄養サマリーの見方

- `枚数`: 読み取ったカード枚数
- `基本栄養`: カロリー、炭水化物、タンパク質、脂質
- `ミネラル・食物繊維`: 食物繊維、カリウム、カルシウム、鉄
- `ビタミン・塩分`: ビタミンA、ビタミンC、ビタミンD、食塩

1枚のときはそのカードの栄養情報が表示され、複数枚のときは各項目の合計が表示されます。

## 6. 手入力テスト

カメラを使わずに動作確認したい場合は、`手入力` に次のような文字列を入力して `追加` を押します。

```text
card001,rice,150,234,55.7,3.8,52,1200,230,45,1.2,310,35,5.5,0.4
```

## 7. CSV取込

カードデータが多い場合は `CSVを選択` からまとめて取り込めます。

対応形式:

```csv
card_id,name,grams,calories,carbs,protein,fat,fiber,potassium,calcium,iron,vitaminA,vitaminC,vitaminD,salt
card001,rice,150,234,55.7,3.8,52,1200,230,45,1.2,310,35,5.5,0.4
card002,bread,80,210,40.2,6.5,48,900,120,25,0.8,0,0,0,0.6
```

- ヘッダーあり・なしのどちらでも取り込めます。
- 同じカードIDがすでに登録済みの場合は重複としてスキップされます。
- 取り込み結果は画面下部のメッセージで件数表示されます。

## 8. データ操作

- `削除`: 対象カードだけを削除します。削除後は同じカードIDを再登録できます。
- `リセット`: 読み取り済みカードをすべて消去します。
- `CSV書き出し`: 現在の読み取り結果を CSV ファイルとして保存します。

## 9. トラブル対応

### カメラが開始できない

- GitHub Pages などの HTTPS URL で開いているか確認してください。
- ブラウザのカメラ許可がブロックされていないか確認してください。
- ほかのアプリがカメラを使用していないか確認してください。

### QRコードを読み取れない

- QRコードを画面中央の枠内に入れてください。
- 画面や紙の反射を避けてください。
- QRコード内の文字列が 15 項目形式になっているか確認してください。

### 失敗音が鳴る

- QRコード自体は読めているが、列数や数値形式が不正の可能性があります。
- `カードID,名称,グラム数,カロリー,炭水化物,タンパク質,脂質,食物繊維,カリウム,カルシウム,鉄,ビタミンA,ビタミンC,ビタミンD,食塩` の並びを確認してください。

### CSV取込でエラーになる

- 列数が 15 項目になっているか確認してください。
- 数値項目に文字列が混ざっていないか確認してください。
- 引用符付き CSV の場合、引用符が途中で閉じ忘れになっていないか確認してください。

### 同じカードが追加されない

これは重複防止機能です。同じカードIDは1回だけ登録されます。再登録したい場合は該当行の `削除` を押すか、`リセット` してください。

## 10. 教材・授業での運用メモ

- 最初は手入力で栄養サマリーの計算ロジックを確認してください。
- 次に少数のCSVを使って一括取込の動作を確認してください。
- その後にスマートフォン画面へ QR コードを表示し、読み取りテストを行ってください。
- 最後に紙へ印刷し、Windows と Chromebook の両方で動作確認してください。
